DESCRIPTION:
------------

JibJob is a Rails application that lets you compose resumes using plain
text, and publish them through web, mobile, and print (PDF export).

REQUIREMENTS:
-------------

* Ruby 2.2
* Rails 4.2
* Postgresql 9.4
* Redis
* Memcache
* wkhtmltopdf
* pdftk
